# project 說明
就是一般的 todo App。
可以新增、刪除、顯示完成、存入 localStorage 、刪除 localStorage。

# key concepts
DOM 操控如增加元素、刪除元素；監聽事件、事件代理；本地端暫存以及刪除。
JavaScript 語法熟悉。

# 預計可加入的 feature
- 可以對 todo 分類。
- 可以對已新增的 todo list 做編輯功能。
- 再簡化程式，例如可以把 addEventListener 弄成 function (參考 https://www.youtube.com/watch?v=XF1_MlZ5l6M&t=1s 16:00)
