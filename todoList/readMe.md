# project 說明
就是一般的 todo App。
可以新增、刪除、顯示完成、存入 localStorage 、刪除 localStorage。

# key concepts
DOM 操控如增加元素、刪除元素；監聽事件、事件代理；本地端暫存以及刪除。
JavaScript 語法熟悉。
